# @bigslycat/prettier

My prettier config.

## Usage

Install:

```sh
npm i -D @bigslycat/prettier prettier
# or
yarn add -D @bigslycat/prettier prettier
```

### Init config

```sh
npx init-prettier
# or
yarn init-prettier
```

### Manual init

```sh
touch prettier.config.js
```

#### prettier.config.js

```js
// @ts-check

export { config as default } from '@bigslycat/prettier'
```
